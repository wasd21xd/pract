Проект уже открыт и готов к работе!

### Шаг 2: Следуйте QUICKSTART.md

## 🔧 Технологический стек

### Frontend
- React 18.2 - самый популярный UI фреймворк
- TypeScript - для типизации и надёжности
- Vite - супер быстрая сборка
- Zustand - лёгкое управление состоянием
- CSS3 - красивые стили

- Node.js - серверный JavaScript
- Express 4.18 - микрофреймворк для API
- Prisma - современный ORM
- PostgreSQL - надёжная БД
- WebSocket - реал-тайм общение
- JWT - безопасная аутентификация
- bcryptjs - защита паролей
