# 🎯 Team Management App

**Полнофункциональное приложение для управления командой с реал-тайм функциональностью.**

> ⚡ **Быстрый старт**: Откройте [QUICKSTART.md](./QUICKSTART.md) для запуска за 5 минут!

## 📋 Что это?

Веб-приложение для управления командой и задачами с:
- 👤 Регистрацией и аутентификацией
- 👥 Управлением командой и приглашениями
- ✅ Управлением задачами (Kanban доска)
- 💬 Реал-тайм чатом через WebSocket
- 📊 Отслеживанием активности

## ⚡ Быстрый старт (5 минут)

```bash
# 1. Убедитесь что установлены:
node -v    # Node 18+
npm -v     # npm 8+
psql -v    # PostgreSQL

# 2. Создайте БД
createdb -U postgres team_management

# 3. Установите зависимости
cd frontend && npm install && cd ..
cd backend && npm install && cd ..

# 4. Инициализируйте БД
cd backend && npx prisma migrate dev --name init && cd ..

# 5. Запустите оба сервера
./run.sh

# 6. Откройте http://localhost:5173
```

**Готово!** Регистрируйтесь и начните использовать приложение.

## 🎯 Функциональность

### ✅ Аутентификация
- Регистрация с email и паролем
- Безопасный вход через JWT
- Хеширование паролей (bcryptjs)

### ✅ Управление командой
- Создание команд с уникальными кодами
- Приглашение членов по коду
- Просмотр членов команды
- Отслеживание активности

### ✅ Управление задачами
- Создание, редактирование, удаление
- Kanban доска (3 статуса: todo, in-progress, done)
- Быстрая смена статуса
- Назначение на пользователей

### ✅ Реал-тайм чат
- WebSocket соединение
- Мгновенные сообщения
- История чата
- Видно кто пишет

## 🧪 Примеры использования

### Создание аккаунта
1. Откройте http://localhost:5173
2. Введите email, пароль, имя
3. Нажмите "Зарегистрироваться"
4. Будет создан JWT токен

### Создание команды
1. На главной странице нажмите "Создать команду"
2. Введите название
3. Получите уникальный код для приглашений

### Добавление задачи
1. Перейдите на вкладку "Задачи"
2. Заполните название и описание
3. Нажмите "+ Добавить задачу"
4. Измените статус через dropdown

### Реал-тайм чат
1. Откройте вкладку "Чат"
2. Введите сообщение
3. Нажмите "Отправить"
4. Сообщение появится в реальном времени

## 🐛 Решение проблем

### Ошибка "Cannot find module"
```bash
npm install
npx prisma generate
```

### PostgreSQL не подключается
```bash
# Проверьте что PostgreSQL запущен
brew services start postgresql  # macOS
sudo systemctl start postgresql # Linux

# Создайте БД если её нет
createdb -U postgres team_management
```

### Порты уже заняты
```bash
# Найдите процесс
lsof -i :3000  # бэкенд
lsof -i :5173  # фронтенд

# Завершите его
kill -9 <PID>
```
