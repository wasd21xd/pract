## 🚀 Как всё это работает

### Архитектура
```
1. Фронтенд (React + Vite)     [localhost:5173]
   ↓
2. API запросы через HTTP      [localhost:3000/api/*]
   ↓
3. Express сервер
   ├─ Аутентификация (JWT)
   ├─ API маршруты (CRUD)
   └─ WebSocket сервер         [ws://localhost:3000]
   ↓
4. PostgreSQL БД
```

### Процесс взаимодействия

**1. Регистрация пользователя:**
```
User вводит email/пароль → React отправляет POST /auth/register
→ Express хеширует пароль → Сохраняет в БД → Возвращает JWT token
→ React сохраняет token в localStorage → Переход на Dashboard
```

**2. Создание задачи:**
```
User нажимает "Добавить задачу" → React отправляет POST /tasks
→ Express проверяет JWT → Сохраняет в БД → Возвращает задачу
→ React обновляет Zustand store → Отображает на доске
```

**3. Реал-тайм чат:**
```
User отправляет сообщение → React подключается к WebSocket
→ Отправляет сообщение серверу → Express broadcasting ко всем клиентам
→ Все клиенты получают сообщение → React обновляет ChatPanel
```
